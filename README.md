## 4. 遇到的问题及解决方法
**问题 1：VS Code 终端提示“无法将 git 项识别为 cmdlet...”**
* **原因**: 系统没有安装 Git 环境，或者环境变量未配置生效。
* **解决方法**: 前往官网下载并安装 Git，随后彻底重启 VS Code，使其加载最新的系统环境变量。

**问题 2：提交时提示 "Author identity unknown"**
* **原因**: 没有配置全局的用户邮箱和用户名，Git 出于协作规范拒绝提交。
* **解决方法**: 运行 `git config --global user.email "我的邮箱"` 和 `user.name "我的名字"` 补充身份信息。

**问题 3：推送代码时遭遇 "Failed to connect to github.com port 443" 超时错误**
* **原因**: 国内网络环境连接 GitHub 存在波动或被屏蔽。
* **解决方法**: 尝试连接手机热点；或灵活变通，改用 GitHub 网页端的 "upload file" 和在线新建文件功能完成最终的作业提交。
